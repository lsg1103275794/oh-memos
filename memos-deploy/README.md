## 中文

基于 [MemOS](https://github.com/MemTensor/MemOS) 的 Windows 便携式部署方案。

### 特点

- 便携式 Python 环境，无需系统安装
- Ollama 本地嵌入模型，降低 API 成本
- 支持 OpenAI 兼容 API
- 一键启动脚本
- 详细的中文文档

### 快速开始

```cmd
# 一键启动
双击 run.bat

# 首次安装
双击 install_run.bat
```

**服务地址**: http://localhost:18000/docs

## English

A Windows portable deployment solution based on [MemOS](https://github.com/MemTensor/MemOS).

### Features

- Portable Python environment, no system installation required
- Ollama local embedding model to reduce API costs
- OpenAI-compatible API support
- One-click startup scripts
- Detailed documentation

### Quick Start

```cmd
# One-click start
Double-click run.bat

# First-time installation
Double-click install_run.bat
```

**Service URL**: http://localhost:18000/docs
