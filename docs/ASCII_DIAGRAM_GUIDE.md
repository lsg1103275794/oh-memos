## Box Drawing 字符速查表

### 基础线条
```
─  横线 (U+2500)
│  竖线 (U+2502)
```

### 角落
```
┌  左上角 (U+250C)
┐  右上角 (U+2510)
└  左下角 (U+2514)
┘  右下角 (U+2518)
```

### T 形连接
```
├  左 T (U+251C)
┤  右 T (U+2524)
┬  上 T (U+252C)
┴  下 T (U+2534)
┼  十字 (U+253C)
```

### 箭头
```
▶  右箭头 (U+25B6)
◀  左箭头 (U+25C0)
▲  上箭头 (U+25B2)
▼  下箭头 (U+25BC)
→  细右箭头 (U+2192)
←  细左箭头 (U+2190)
↑  细上箭头 (U+2191)
↓  细下箭头 (U+2193)
```

### 双线框
```
╔══╗
║  ║
╚══╝
```

### 圆角框
```
╭──╮
│  │
╰──╯
```

## 常用模板

### 简单方框
```
┌─────────────┐
│   Title     │
├─────────────┤
│  Content    │
└─────────────┘
```

### 流程图
```
┌───────┐     ┌───────┐     ┌───────┐
│ Start │────▶│Process│────▶│  End  │
└───────┘     └───────┘     └───────┘
```

### 层级结构
```
┌─────────────────────────┐
│         Parent          │
├─────────────────────────┤
│  ┌─────────┐ ┌─────────┐│
│  │ Child 1 │ │ Child 2 ││
│  └─────────┘ └─────────┘│
└─────────────────────────┘
```

### 树形结构
```
Root
├── Branch 1
│   ├── Leaf 1.1
│   └── Leaf 1.2
├── Branch 2
│   └── Leaf 2.1
└── Branch 3
```

### 时序图
```
  Client                Server
    │                     │
    │  1. Request         │
    │────────────────────▶│
    │                     │
    │  2. Response        │
    │◀────────────────────│
    │                     │
```

## Python 生成器

运行项目自带的生成器：

```bash
python tools/generate_ascii_diagram.py
```

生成的图表：
- `architecture` - MemOS 技术栈
- `memory_flow` - 记忆流程图
- `memory_cube` - 记忆立方体结构
- `mcp_flow` - MCP 交互流程

## 视频制作建议

### 动画效果

1. **逐行显示**: 用视频剪辑逐行显示 ASCII 图
2. **打字机效果**: 模拟终端打字效果
3. **高亮闪烁**: 重点部分用不同颜色高亮

### 推荐字体

| 字体 | 说明 |
|------|------|
| **JetBrains Mono** | 编程字体，等宽 |
| **Fira Code** | 带连字的编程字体 |
| **Source Code Pro** | Adobe 出品 |
| **Consolas** | Windows 默认 |

### 配色方案

```
背景: #1E1E1E (深灰)
文字: #D4D4D4 (浅灰)
高亮: #569CD6 (蓝色)
重点: #4EC9B0 (青色)
警告: #CE9178 (橙色)
```
